## 📝 Task-Tracker
Keep your workflow organized with this free task tracking web app.
Arrange, Create, Delete any task any time 😁. This project is free to use for whatever purpose

# 🧠 Features
- Fully offline
- Saves your tasks to local storage
- Really easy to use
- Fully customizable

# 👨🏻‍💻 Languages Used

- JavaScript / ES6
- CSS 3
- HTML 5
